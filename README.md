# Automotive-Spare-Parts-Business-Analytics
Business Intelligence &amp; Data Analytics project for an automotive spare parts retailer using Power BI, Excel, Power Query, and Python (Pandas).

## Project Overview

This project presents a complete Business Intelligence and Data Analytics solution developed for an automotive spare parts business.

The objective was to transform raw operational and financial data into meaningful business insights that support management in monitoring sales performance, inventory movement, customer balances, purchasing activities, and profitability.

The project combines Excel, Power BI, and Python (Pandas) to perform data preparation, analysis, KPI development, and interactive dashboard reporting.

---

## Business Problem

The client managed daily business transactions using multiple Excel worksheets without a centralized reporting solution.

As a result, management faced challenges in:

* Monitoring daily and monthly sales performance.
* Tracking customer outstanding balances.
* Managing inventory levels efficiently.
* Evaluating supplier purchasing activities.
* Measuring profitability across products and transactions.
